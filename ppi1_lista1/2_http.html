O que é o HTTP?

O HTTP é o protocolo que faz a comunicação entre o navegador e o servidor da WEB.

Quando você entra em um site, o navegador envia uma requisição HTTP e o servidor dá uma resposta HTTP (A página).

Fluxo

  1. você digita URL
  2. O navegador envia um request
  3. O servidor envia um response
  4. A pagina aparece bonitinha (ou não) pra você visualizar

Métodos mais utilizados na web: GET e POST

GET: Busca dados do servidor. Apenas consulta (não envia). Fica na URL.

POST: Pode alterar dados no servidor. Como por exemlo usuário e senha, formulários. Fica oculto no corpo do site.


URL: https://www.google.com/search?q=brasilzao

Host: https://www.google.com

GET: search?q=brasilzao

Códigos de status mais importantes

  • 200 -> OK(ocorreu corretamente)
  • 404 -> Página não encontrada
  • 500 -> Erro no servidor
