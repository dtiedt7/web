História da Web

A web é um serviço criado pelo Tim Barnes Lee, em 1990. Que funciona sobre a internet, que é uma rede de computadores, utilizando a arquitetura Cliente - Servidor

Para isso ele definiu a estrutura de funcionamento da web, cliente servido e criou a imagem HTML, para exibição de documentos e o protocolo HTPP para comunicação entre aplicação cliente e servidor.

Para cuidar dos padrões da web e propriedade da tecnologia ele criou a fundação w3c World Web Consorcium

A referência das tags HTML pode ser encontrada no site da w3c html reference
