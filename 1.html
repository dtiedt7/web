O que é HTML?

HTML é uma linguagem usada na internet para montar páginas. Ele não serve para programar, mas para organizar textos, imagens e outras coisas na tela.

O papel do HTML na Web é criar a estrutura dos sites, ou seja, ele mostra para o navegador como o conteúdo deve aparecer.

  • Organiza o conteúdo
  • Permite colocar textos e imagens
  • É a base dos sites
